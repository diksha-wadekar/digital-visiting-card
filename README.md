# Digital Visiting Card

An interactive flip-card digital visiting card built with pure HTML and CSS. Clicking/tapping the card flips it between a college ID layout and a personal contact card using CSS 3D transforms.

## Features
- 3D flip animation on click/tap (no JavaScript required)
- Front: institute ID card layout (name, department, roll no., batch, validity)
- Back: personal contact card (name, role, email, phone)
- Fully responsive, works on mobile and desktop

## Tech Stack
- HTML5
- CSS3 (3D Transforms, Transitions)

## How to Run
1. Clone the repo
2. Open `index.html` in any browser
   or
3. View the live demo: [Live Demo]([your-github-pages-link](https://diksha-wadekar.github.io/digital-visiting-card/))
